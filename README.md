### 01. PHP webshell 概述

**一句话 webshell**

- [php/01.php](php/01.php) -- Single Line Webshell

    ```Plaintext
    Description:
        PASSWORD: krm
        MIME Type: text/php
        Request Method: $_REQUEST

    Args:
        MIME Type:
            image/jpeg
            image/png
            text/plain
        Request Method:
            $_GET
            $_POST
        Suffix:
            
    ```

**配置文件覆盖**

- [php/.htaccess](php/.htaccess)

    ```Plaintext
    Description:
        Parse ".l33t" suffix file as PHP file
        MIME type: text/plain
    ```
